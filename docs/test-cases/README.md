# Test Cases

Thư mục này chứa các Test Cases cho Cinema Booking System.

## Files

- `UTC_Unit_Test_Cases.md`: Unit Test Cases (~60-80 TC)
- `ITC_Integration_Test_Cases.md`: Integration Test Cases (~40-50 TC)
- `STC_System_Test_Cases.md`: System Test Cases (~50-60 TC)

## Test Case Format

Mỗi test case bao gồm:
- Test Case ID
- Test Name
- Module/Feature
- Priority
- Preconditions
- Test Steps
- Expected Results
- Test Data

## Execution

- Unit Tests: `mvn test`
- Integration Tests: `mvn verify` hoặc Postman/Newman
- System Tests: Manual execution hoặc automated UI tests
